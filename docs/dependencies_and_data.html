Required software and data¶

Software¶

To install all required packages it should be sufficient to simply install the aws python package using pip:

```

pip install -e .

```

Data¶

Scattering data¶

We currently have two version of the scattering data that we are using, the default so-called standard habits and a subsampled version of them. They are available from my home directory.

For the original data:

cp -r /home/simonpf/src/aws/data/StandardHabits ~/aws/data

For the sub-sampled data for faster simulations:

cp -r /home/simonpf/src/aws/data/StandardHabits_small ~/aws/data/

The aws package is currently configured to use only the data in StandardHabits_small. So the original data is included here only for completeness.

TELSEM atlases¶

To copy the atlases containing the surface emissivities run

cp  /home/simonpf/src/aws/data/ssmi_*_M2 ~/aws/data/
cp  /home/simonpf/src/aws/data/correlations ~/aws/data/
